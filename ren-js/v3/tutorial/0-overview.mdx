This tutorial will walk you through writing a front-end app that uses RenVM to mint BTC to an Ethereum smart contract. It is designed to be a quick introduction to RenVM and should be read together with the other examples, which provide more in-depth explanations.

Sections:

1. [Smart Contract](./tutorial/smart-contract) - Write a basic smart contract that can mint renBTC
2. [React Setup](./tutorial/react-setup) - Setup the React repo
3. [Chain Transactions](./tutorial/chain-transactions) - Write a component for handling chain transactions
4. [Gateway Transactions](./tutorial/gateway-transactions) - Write a component for handling gateway transactions
5. [Gateways](./tutorial/gateways) - Handle creating new gateways
6. [App](./tutorial/app) - Create the top-level App component

<br />

:::tip

The code for this tutorial can be found on GitHub: <https://github.com/renproject/ren-js-v3-tutorial>, and a live version can be found here: <https://renproject.github.io/ren-js-v3-tutorial>
