assertType#

â–¸ Const assertType<T>(type: string, objects: { [value: string]: T; }): objects is object

The following is a set of rudimentary type validation functions.

The main function is assertType, which accepts a type and a dictionary of values.

The type must be a string that matches the following pattern:

TYPE:
| TYPE '|' TYPE
| Array<TYPE>
| TYPE[]
| PRIMITIVE_TYPE
PRIMITIVE_TYPE:
| "string"
| "number"
| "bigint"
| "boolean"
| "symbol"
| "undefined"
| "object"
| "function"
| "null"
| "any"
| "Buffer"
| "BigNumber"

Types are matched by a regex so '|' can't be used at multiple levels, e.g. string | Array<string | number>.
