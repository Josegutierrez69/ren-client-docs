The following example shows how to query the latest epochs.

<GraphQL
  url="https://api.thegraph.com/subgraphs/name/renproject/renvm"
  defaultQuery={`{
  epoches(
    orderBy: timestamp,
    # Start from the first public epoch.
    skip: 16
  ) {
    numberOfDarknodes
    timestamp
    blockNumber
  }
}`}
/>
