DarknodeSlasher

Reference

Contract: DarknodeSlasher#

DarknodeSlasher will become a voting system for darknodes to deregister other misbehaving darknodes. Right now, it is a placeholder.

Inherits from:

  • Claimable
  • Ownable
  • Context
  • Initializable
