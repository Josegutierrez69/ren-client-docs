### **`remove`** *`(struct LinkedList.List self, address node)`* (internal)

**Params**
- `self`: The list being using.

- `node`: The node in the list to be removed.


Remove a node from the list, and fix the previous and next
pointers that are pointing to the removed node. Removing anode that is not
in the list will do nothing.
