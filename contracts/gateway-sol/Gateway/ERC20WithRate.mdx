## Contract: `ERC20WithRate`

ERC20WithRate allows for a more dynamic fee model by storing a rate
that tracks the number of the underlying asset's unit represented by a
single ERC20 token.

Inherits from:

-   `ERC20`
-   `IERC20`
-   `Ownable`
-   `Context`
-   `Initializable`
