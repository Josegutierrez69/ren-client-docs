RenVM Networks

RenVM has two networks available to developers - Mainnet and Testnet.

Testnet RenVM can be used to develop and test using testnet funds. Any chains supported by RenVM Testnet are also testnet chains, e.g. Bitcoin Testnet, and multiple Ethereum testnets (Kovan and GÃ¶rli). Each of these testnet chains have their own docs and faucets available online.

Testnet RenVM has its own REN token (see contract deployments for the address). If you require testnet REN for testing registering darknodes on testnet, please contract the Ren team.

A list of links and resources for each of RenVM's networks can be found below:

  • Command Center - for RenVM stats and for managing darknode
    https://mainnet.renproject.io
  • RenBridge - the official interface for minting and burning ren-assets
    https://bridge.renproject.io
  • Dev Tools / Explorer - for looking up RenVM transactions
    https://explorer.renproject.io/
  • RPC - JSON-RPC endpoint - see the JSON-RPC tab
    https://rpc.renproject.io/
  • Stats - GraphQL endpoint for querying RenVM stats - see the Stats APIs tab
    https://stats.renproject.io/
